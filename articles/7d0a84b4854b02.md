## 知的財産

知的財産とは、人間の知的な活動によって生み出された、形のない財産のことである。
代表的なものとして以下3つがある。

| 種類 | 保護対象 | 例 |
| ---- | ---- | ---- |
| 著作権（Copyright） | 表現 | 音楽、イラスト、ソースコードなど |
| 特許権（Patent） | 仕組み、アイデア | 技術、システムなど |
| 商標権（Trademark） | 名前やマーク | ブランドのロゴ、社名など |

米国の著作権法では、死亡してから70年、公表されてから95年、作成してから120年で著作権が失効する。いずれかの条件を一番最初に満たした時点で失効する。

ロゴの上に表示されているRのマーク🄬が商標であることを示すためのマークである。Cのマーク🄫は著作権。

:::message
知的財産が保護されてない場合、他人の発明を模倣して自分の利益にすればよいだけなので、技術が発展しなくなってしまう。
:::

### 不正競争の防止

不正競争の防止とは、他人の成果や信用を盗んで利益を得る行為を禁止し、公正な競争を守るためのルール。
他人の商品の形態を模倣した商品を譲渡等する行為や、営業秘密の侵害、ドメイン名の不正取得などがそれにあたる。

### ライセンス

アプリケーションやソフトウェアなどの知的財産の利用権利である。

:::message
アプリなどを一度購入すれば、スマホを買いなおしてもアプリを買いなおす必要はない。これはアプリの利用権利を購入しているからである。
:::

### 消尽

特許製品を他人へ譲渡した場合、その製品についての特許権はもう主張できない。ライセンスに関しては、販売ではなく権利を貸し出しているだけのため、消尽しない。

:::message
正規に購入した製品の転売は問題ないが、コピー商品の販売は著作権侵害である。
:::

## デジタルミレニアム著作権法

デジタルコンテンツの著作権に関する法律である。自分の著作物が、他人によって勝手に利用、転載などされた場合に削除を要求することができる。削除を要求する場合は、サイトの運営者に権利侵害された旨を通知し、サイト運営者がコンテンツを削除します。削除された場合は、著作権侵害の責任が免除される。

### 電子透かし

デジタルデータの中に、識別子情報を埋め込む技術のことである。基本的には人間に見えない形で埋め込まれるが、発売前の商品の画像などに目に見える形で表示される「SMAPLE」という文字も電子透かしの一種である。
電子透かしを埋め込む目的は、著作権保護や、不正コピーの追跡などである。

## EEA（連邦経済スパイ法）

企業秘密を保護するための法律。刑事法のため、不正に企業秘密を取得、漏洩した外国の企業や政府には罰則がある。

:::message
民法にもEEAと似たトレードシークレット法という州法がある。
:::

## まとめ

- 知的財産は企業の大事な資産であり保護する必要がある
- デジタルミレニアム著作権法はデジタルコンテンツのを守るための法律
- EEAは企業秘密を守るための法律
