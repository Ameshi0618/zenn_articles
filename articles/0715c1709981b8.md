## 身体情報のプライバシー

プライバシーは守るべきプライバシーごとに法律があり、病気などの身体情報もプライバシーとして扱われる。

![](https://storage.googleapis.com/zenn-user-upload/cc2e1a88ed5a-20260127.png)

### HIPAA（医療保険の相互運用性と責任に関する法律）

電子化した医療情報に関するプライバシー保護・セキュリティ確保について定めたアメリカの法律である。
個人の健康状況や支払い状況などの個人を特定できる保護対象保険情報をPHI（Protected Health Information）を保護するのが目的である。
PHIの中でもデジタル形式のPHIについてはePHIという。

| 項目 | PHI（Protected Health Information） | ePHI（electronic PHI） |
| ---- | ---- | ---- |
| 定義 | 個人を特定できるすべての健康情報| 電子的な媒体で保存・送信されるPHI |
| 形式 | 紙、口頭、電子データ | 電子データ |
| 例 | 診察メモ、電話の会話、診察券 | 電子カルテ、メール |

### HITECH（経済的および臨床的健全性のための医療情報技術に関する法律）

HITECHはHIPAAの後に制定された法律で、特にITベンダーへの責任拡大や、電子化に伴うセキュリティ侵害への罰則を大幅に強化したものである。

HITECH制定前は、患者は自分の個人データが誰に開示されたかを知ることができなかったが、HITECH制定以降は、開示内容の要求ができるようになった。
開示内容が要求された場合は、誰がどんな権限で個人データにアクセスしたのか等を説明する必要がある。

::: message
HIPAAは医療情報の保護に関する米国の法律であり、プライバシー規則とセキュリティ規則の両方を規定しています。PHIはもともとは、ただの概念であったが、2つの規則が成立する過程で、正式にPHI、ePHIの順で言葉が定義され、ePHIを強化する形でHITECHが制定された。
:::

## 児童のプライバシー

プライバシーが保護される権利は大人だけでなく、子供にもその権利がある。

### COPPA（児童オンラインプライバシー保護法）

13歳未満の子供の個人情報をオンラインサービス（アプリ、ゲームなど）から守るための法律である。
13歳未満の子供から個人情報を収集する際には、保護者の同意を必ず得る必要がある。

::: message
YouTubeなどは13未満の子供も利用するため、児童オンラインプライバシー保護法が適用される。いろんな人に使ってもらうサービスを作ろうとすれば、必ず利用者に子供も含まれてくる。その際には法律に違反しないように対象年齢などには十分注意を払う必要がある。
:::

#### YouTubeのCOPPA違反

2019年、YouTubeが子供向けチャンネルの視聴者から、保護者の同意なしにデータを収集し、パーソナライズ高校を表示していたとして**1億7000万ドル（約180億）**の制裁金が科された。

#### Epic Games（Fortnite）のCOPPA違反

2022年、Fortniteの音声・テキストチャットがデフォルトで有効になっていたことが問題視されたことで、**5億2,000万ドル（約710億）**の制裁金が科された。

## まとめ

- HIPAAとHITECHは身体情報のプライバシーを保護する法律
- COPPAは子供のプライバシーを守る法律
