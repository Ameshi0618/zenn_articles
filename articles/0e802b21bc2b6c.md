## 証拠の種類

証拠は大きく以下の3つに分類することができる

::: message
分類方法は様々あるので、別の分類方法も存在する
:::

| 種類 | 内容 | 例 |
| ---- | ---- | ---- |
| 物証 | 物の状態や性質から事実を証明する | ウイルスが仕込まれたUSB、破壊されたPC |
| 書証 | 書面の記載内容を証拠とする | ログ、電子メールの内容 |
| 人証 | 人の供述を証拠とする | 目撃談、専門的意見 |

CISSPとして法廷に立つ場合、専門的意見として人証として証拠能力を持つ。

## デジタル証拠の完全性（証拠保全）

デジタル証拠は、「改ざんされていないこと」を証明することが重要になる。

- ハッシュ値：証拠を収集したらすぐ、そのデータのハッシュ値などを計算し記録する。
- オリジナルの保存：オリジナルから複製したデータに対して解析などを行う

::: message
インシデント対応などが発生した場合は、証拠保全とシステム復旧で対立することがある。あらかじめ、インシデント対応計画の中で、保全と復旧のどちらを優先するか決めておくとよい。
:::

## 証拠の優先順位

証拠には優先順位が存在する

### 最良証拠の原則

コピーされたデータとオリジナルのデータが異なる場合、オリジナルのデータを優先される

### 伝聞証拠禁止の原則

人から聞いた話などは、証拠にはならないという原則。勘違いや、嘘である可能性があることから正確性を検証できないからである。

::: message
デジタル証拠は伝聞証拠なのか
→コンピュータが自動で生成したログなどは伝聞証拠ではない。ただし、人間が作成したデータに記載されている意見や主張は原則伝聞証拠とみなされる。時刻を同期したうえでログを取ることが非常に重要である。
:::

### 口頭証拠排除の原則

契約書に関する原則であり、口頭での合意内容よりも書面が優先される

## 合衆国憲法修正第4条

不当な捜索及び押収から国民を保護するための法律。
セキュリティ専門家、フォレンジック担当者にとって、デジタル証拠を収集するうえで重要な法律である。

条文の要旨
```
人民が、その身体、住宅、書類および所有物について、
不当な捜索および押収を受けることなく、安全を保つ権利は、侵してはならない。
いかなる捜査令状も、相当な理由に基づいて発行されなければならない。
```

- 不当な捜索の禁止：正当な理由なく個人のPC、スマートフォン、クラウドストレージなどを調査することはできない
- 相当な理由：令状を取得するには、犯罪の証拠があると信じるに足る具体的な証拠が必要である
- プライバシーの保護：曖昧な理由での調査はプライバシー侵害の恐れがある

### 排除法則

合衆国憲法修正第4条に違反して入手された証拠は、裁判において証拠として認められないというルール。また、不当な捜索によって見つかったヒントから得られた二次的証拠もすべて無効になります。


:::message
2014年にあったRiley v. Californiaの事例で、交通違反で逮捕した人の所持品から令状なしにスマホを押収し、そこから見つかった証拠によって殺人未遂罪として起訴された。しかし、合衆国憲法修正第4条に違反したとし、排除法則が適用され有罪判決が取り消された。
:::

## まとめ

- 証拠には優先度があり、物証が一番優先される
- 不当な捜索、押収は禁止され、違法に得た証拠は証拠能力を持たない
- デジタル証拠は、改ざんされていないことの証明が重要
