## ライフサイクル

データのライフサイクルとは、データが生成または収集、されてから、最終的に破棄されるまでの一連の流れのことを表す。

| ステップ | 概要 | 補足 |
| ---- | ---- | ---- |
| 生成・収集 | ログ、ユーザーの入力、外部APIなどからデータが作成または取得される |  |
| 配布・保管 | 収集したデータをデータベース、クラウドなどに保存 |  |
| 利用 | アプリケーションでの処理や、分析のためにデータが読み込まれ加工される |  |
| 保持 | 業務利用から外れた場合でも、将来的な監査などのために長期保持する |  |
| 破棄 | 保持期間が過ぎた場合や、不要になったデータを復元不可能な状態にする |  |


:::message
ライフサイクルに従って正しくデータを扱えていない場合、どのデータが最新で正しく、どのデータが古く不要なのか判断ができなくなる。そうなってしまえば、正しいデータが含まれていたとしても使えないデータとなってしまう。
:::
