### SOX法（上場企業会計改革および投資家保護法）

企業の粉飾決算を防ぎ、投資家を守るための法律である。内部統制を強化することで財務情報の正しさを証明し、投資家の信頼を確保することが目的である。

2000年代にアメリカの企業が大規模な粉飾決算を行い、倒産した。これにより投資家は巨額の損失を被り、市場の信頼が失墜したという背景があり、2002年に米国連邦法として制定された。

:::message
J-SOX法という、SOX法を日本に適用できるように改善された法律がある
:::

- 完全性の重視：SOX法において最も重要なのがCIAのうちの「I（完全性）」である。財務情報が改ざんされてないことをログやハッシュで証明する必要がある
- 職務分離：会社とは独立した公認会計士や、社外取締役を選任し監査してもらう必要がある

---

### GLBA（グラム・リーチ・ブライリー法）

金融機関の、顧客のプライバシーとデータ保護に関する法律。

元々アメリカでは、グラススティーガルによって銀行業と証券・保険業は厳格に分けられていたが、GLBAによってこの垣根が取り払われ、金融の巨大化が進んだ。その際に、「データの安全管理」も法律に組み込んだ。データが一か所に集約するようになったことで、より厳格なセキュリティが求められるようになった。

:::message
この法律に個人顧客に対する救済措置は含まれておらず、情報漏洩した場合でも個人が金融機関を訴えることはできません。よって被害にあった際は、他の法律による救済措置を求める必要がある。
:::

## 国家安全保障・通信関連

### PATRIOT法（米国愛国者法）

アメリカの同時テロの発生直後に、**テロ対策**のためにわずか45日という異例の速さで成立した連邦法である。

主に以下3つの活動の拡大を図るための法律である

| 項目 | 内容 | 
| ---- | ---- |
| 監視権限の拡大 | 電話、メール、インターネット履歴の傍受が容易になった |
| 情報共有の緩和 | 捜査機関（FBI）とインテリジェンス機関（CIA）のコミュニケーションの緩和 |
| 罰則の強化 | テロ犯罪に対する罰則の強化と、テロ容疑対策にかける活動の拡大 |

:::message
監視権限の拡大によってプライバシー侵害の恐れがあるが、PATRIOT法はこれに例外を設ける形になった。
:::

政府がテロ調査に必要と判断した場合は、裁判所の令状なしに企業の顧客データを押収することが可能になった。

:::message
査官にいきなり企業の顧客データの提出を求められた場合、CISSPとして理性的な行動を心掛ける必要がある。情報を提供した後、それが不当な捜索であったとわかった場合、(ISC)²倫理規約の「合法的な行動」に反し、さらに「顧客への守秘義務違反」で訴えられるリスクを背負うことになる。
:::


### CALEA（法執行のための通信援助法）

捜査機関が合法的な通信傍受を行うために、通信事業者に協力を義務付けた法律である。

### FISMA（連邦情報セキュリティマネジメント法）

米国邦政府機関に対して、情報および情報システムのセキュリティを強化するためのプログラムを開発、文書化、実践することを義務付けた法律である。
FISMAの要件を満たすための企画としてNIST SP 800-37（RMF）がある。

連邦政府関連の組織は毎年、行政管理予算局（OMB）にセキュリティ報告書を送る必要がある

## まとめ

- SOX法は企業の粉飾決算を防ぎ、投資家を守るための法律
- GLBAは金融機関の、顧客のプライバシーとデータ保護に関する法律
- PATRIOT法はテロ対策のための活動拡大に関する法律
- CALEAは通信会社の傍受協力のための法律
